# 🚗 IntelliPark – Smart Parking Management System

IntelliPark is a **Flask-based Smart Parking Management System** designed to simplify parking operations through a user-friendly web interface. The application enables users to register, log in, book parking spaces, view available parking slots, and complete payments efficiently. It demonstrates the integration of Python, Flask, HTML, CSS, JavaScript, and Bootstrap to build a responsive web application.

---

## 📌 Features

* 🔐 User Registration and Login
* 🚘 Vehicle Booking System
* 🅿️ Real-Time Parking Slot Availability
* 📍 Parking Slot Selection
* 💳 Payment Module
* 📱 Responsive User Interface
* 🧭 Easy Navigation Across Pages

## 🚀 Installation

### 1. Clone the Repository

```bash
git clone https://github.com/your-username/intellipark-smart-parking-system.git
```

### 2. Navigate to the Project Folder

```bash
cd intellipark-smart-parking-system
```

### 3. Install Dependencies

```bash
pip install flask
```

### 4. Run the Application

```bash
python app.py
```

### 5. Open in Browser

```text
http://127.0.0.1:5000/
```
